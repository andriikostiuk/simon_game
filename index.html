Rainbow Master is version of famous Simon Game.
    Initially the device has four colored buttons, each producing a particular tone when it is pressed. You can make it hard choosing 5, 6 or 7 colors difficulty. A round in the game consists of the device lighting up one or more buttons in a random order, after which the player must reproduce that order by pressing the buttons. As the game progresses, the number of buttons to be pressed increases.
    You win if you repeat without mistake sequence of 20 steps.
    Also you can choose between normal and strict mode. With normal mode in case if you press the wrong button same sequence will repeat again. In strict mode if you press the wrong button you start from the beginning.
